Arrow Master

📅 2023년 1월

두 플레이어가 한 화면에서 번갈아 화살을 쏘는 턴제 대전 게임입니다. 마우스 드래그로 방향·세기를 조절하며 바람이라는 랜덤 변수가 게임에 긴장감을 더합니다.

주요 구현 내용

  • 마우스 드래그로 발사 방향·파워 계산 (벡터 이용)
  • 포물선 물리 시뮬레이션 (중력 + 바람 속도 적용)
  • 매 턴마다 바람 방향·세기 랜덤 변경
  • 부위별 히트박스 — 머리(고데미지), 몸통, 다리
  • 체력바 UI 및 플레이어 순서 표시
  • 스틱맨 캐릭터를 도형 조합으로 직접 그리기

더 인기, 더 팔로워

📅 2023년 5월

두 인물의 사진을 보고 SNS 팔로워가 더 많은 쪽을 맞추는 퀴즈 게임입니다. Python의 tkinter로 GUI를 구현하고 pandas로 팔로워 데이터를 관리했습니다.

주요 구현 내용

  • tkinter로 2컬럼 레이아웃 GUI 구성
  • pandas로 CSV 팔로워 데이터 로드 및 비교 로직
  • 21명의 인물 사진(PNG)과 이름·팔로워 데이터 정리
  • 랜덤 두 인물 선택 (중복 방지)
  • 정답·오답 시 팝업 메시지 처리

내 과일을 받아줘

📅 2024년 6월

하늘에서 떨어지는 과일을 소쿠리로 받아치는 5단계 구성의 캐주얼 게임입니다. p5.js로 제작했으며 캐릭터 걷기 애니메이션도 직접 구현했습니다.

스테이지 구성

  • 1단계 사과 5개 — 15초 버티기
  • 2단계 포도 7개 — 20초 버티기
  • 3단계 바나나 10개 — 25초 버티기
  • 4단계 딸기 13개 — 30초 버티기
  • 5단계 수박 15개 — 35초 버티기

주요 구현 내용

  • 좌우 방향키로 캐릭터 이동
  • 캐릭터 발이 교차되는 걷기 애니메이션
  • 과일별 낙하 속도·개수 다르게 설정
  • 소쿠리 충돌 시 과일 튕겨내기 (반사 물리)
  • 성공/실패 화면 및 스테이지 전환
